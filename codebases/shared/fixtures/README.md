# Shared DEMA Fixtures

These fixtures are copied into each maintained codebase so they remain
synchronized while still being kept in separate repositories/folders.

- `benchmark-small`: quick parity checks.
- `benchmark-medium`: runtime benchmark target.

File format:
- `nodes.tsv`: `id`, `fc`, `set`.
- `edges.tsv`: `source`, `target`, `ic`, `ec`.
